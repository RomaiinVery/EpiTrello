# EpiTrello

**EpiTrello** is a comprehensive project management solution inspired by Trello, enhanced with developer-centric features like GitHub integration, Markdown support, and AI automation.

## Repository Structure

-   **[`docs/`](./docs)**: Comprehensive documentation, including architecture, API references, and features.
-   **[`epitrello/`](./epitrello)**: The main application source code (Next.js 15, Prisma, TailwindCSS).

-   **Kanban Boards**: Drag-and-drop task management.
-   **Developer Tools**: GitHub integration, Markdown descriptions.
-   **Advanced Features**: Gantt charts, AI Automation, Real-time updates.
