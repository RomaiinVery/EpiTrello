# EpiTrello

**EpiTrello** is a modern, collaborative Kanban-style project management application built with the latest web technologies.

## Quick Start

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Setup the database**:
    ```bash
    npx prisma generate
    npx prisma migrate dev
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
